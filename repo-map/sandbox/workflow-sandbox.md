## リポジトリ情報: workflow-sandbox

**説明**: パイプラインテスト検証用のサンドボックス・リポジトリ (TypeScript + CLI構成)

### 技術スタック
- Node.js
- TypeScript
- npm

### 主要ディレクトリレイアウト
- `src`: コアソースコードディレクトリ
- `tests`: テストコードディレクトリ

### 設計規約 / ガードレール
- テストは Vitest / Jest などの標準ツールを使用すること。
- コミットメッセージは Git 規約 (Semantic Commit: feat, fix, docsなど) に従うこと。
